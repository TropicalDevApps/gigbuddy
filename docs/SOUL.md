# Project Soul

## Vision
To be the definitive real-time command center for live musicians, eliminating stage friction by synchronizing every band member's view and control.

## Principles
1. **Zero-Lag Synchronization:** Stage state (active song, lyrics scroll) must be near-instant across all devices.
2. **Resilience over connectivity:** Essential stage functions (lyrics/chords) must persist through venue Wi-Fi failures.
3. **Inclusive Integration:** Work with the tools musicians already use (Spotify, YouTube, standard MIDI).
4. **Information Density for the Stage:** High-contrast, glanceable data that keeps the musician in the moment.
5. **Secure Collaboration:** Granular control over who can trigger playback or edit the setlist in real-time.

## Values
- **Reliability:** Built for the "show must go on" mindset.
- **Craftsmanship:** Clean, modern, and type-safe codebase (TS/React).
- **Empowerment:** Tools that make a local band feel like a stadium act.
- **Focus:** Minimizing UI noise to maximize musical performance.
